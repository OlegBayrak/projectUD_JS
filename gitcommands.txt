git add -A добавить все файлы в индекс (они трекаются)
git commit -a -m "Message" - сделать коммит с сообщением
git status - проверить статус репозитория
git push - запушить в ветку
git init - добавить папки гита в проект
git pull - пушнуть изменения последние
git checkout - проверить на какой бранче сейчас 
git checkout test_branch - Переключиться на тест бранчу
git rebuild main - сделать ребилд в тек бранчу из мейна
